# 星脈のレジスタンス

ATBバトル搭載のHTML5ブラウザJRPG。

## 技術スタック

- **Phaser 3** — ゲームエンジン
- **TypeScript** — strict mode
- **Vite** — ビルドツール
- **Canvas 2D API** — キャラクター描画（外部画像不要）
- **Web Audio API** — BGM/SE生成（外部音声不要）
- **GitHub Pages** — デプロイ先

## 開発コマンド

```bash
# 開発サーバー起動
npm run dev

# プロダクションビルド
npm run build

# ビルド結果プレビュー
npm run preview
```

## ゲーム概要

惑星テラノヴァを舞台に、五人のレジスタンスが星脈エネルギーを搾取する巨大企業クロノスに立ち向かう物語。

## 操作方法

### タイトル画面

| 操作 | 動作 |
|------|------|
| 「はじめから」クリック | 新規ゲーム開始 → パーティ編成画面へ |
| 「つづきから」クリック | セーブデータをロードして続きから |

### パーティ編成画面

| 操作 | 動作 |
|------|------|
| キャラクターカードをクリック | パーティメンバーの選択/解除（3人選択） |
| 「出撃」ボタン | 選択した3人で探索パートへ |

### 探索パート（フィールド）

| 操作（タッチ / マウス） | 操作（キーボード） | 動作 |
|------|------|------|
| ◀ ボタン長押し | ← / A キー | 左に歩く |
| ▶ ボタン長押し | → / D キー | 右に歩く |
| 宝箱をタップ | 宝箱をクリック | アイテム入手 |
| セーブポイント（緑の光）をタップ | セーブポイントをクリック | HP/SP全回復 + セーブ |
| ボスボタン（⚔マーク）をタップ | ボスボタンをクリック | ボス戦開始 |
| ☰ ボタン（右上） | — | パーティ編成画面へ |

- 一定歩数ごとにランダムエンカウントが発生します
- キャラクターは歩行アニメーション付きで移動します

### バトル（ATBシステム）

バトルは **ATB（アクティブタイムバトル）** 方式です。各キャラクターのATBゲージが時間経過で溜まり、満タンになると **「READY」** と表示され、コマンド選択が可能になります。

#### バトルの流れ

1. **待機** — ATBゲージが溜まるのを待つ（画面右側の黄色バー）
2. **READY表示** — ゲージが満タンになるとコマンドメニューが表示される
3. **コマンド選択** — 6つのコマンドから行動を選ぶ
4. **ターゲット選択**（攻撃/スキル時） — 敵/味方の上に金色の円が出るので、攻撃対象をクリック
5. **行動実行** — ダメージ演出後、次のキャラのATBゲージ待ちに戻る
6. **繰り返し** — 敵全滅 or 味方全滅まで

#### コマンド一覧

| コマンド | 効果 | 操作手順 |
|------|------|------|
| **たたかう** | 通常攻撃 | コマンド選択 → 敵をクリック |
| **星術** | SP消費スキル（攻撃/回復/補助） | コマンド選択 → スキル一覧から選択 → ターゲット選択 |
| **星晶技** | 星晶石スキル | コマンド選択 → 同上 |
| **アイテム** | ポーション等の使用 | コマンド選択 → アイテム選択 → 味方をクリック |
| **覚醒** | 覚醒ゲージMAX時のみ使用可能。強力な必殺技 | コマンド選択のみ（即発動） |
| **防御** | 被ダメージ半減 + ATBゲージ回復速度UP | コマンド選択のみ（即発動） |

#### バトル画面の見方

```
┌─────────────────────────────────┐
│          敵キャラ（上段）           │
│        名前 + HPバー（赤）         │
│                                   │
│         味方キャラ（中段）          │
│                                   │
│ ┌──────────────────────────────┐ │
│ │ 名前   HP:450/450   ■■■■■■  │ │  ← ATBゲージ（黄）
│ │        SP:120/120   ■■■■■   │ │  ← 覚醒ゲージ（金）
│ │        "READY"               │ │  ← 行動可能表示
│ └──────────────────────────────┘ │
│ ┌──────────────────────────────┐ │
│ │ たたかう │  星術    │          │ │
│ │ 星晶技  │ アイテム │          │ │  ← コマンドメニュー
│ │  覚醒   │  防御   │          │ │
│ └──────────────────────────────┘ │
└─────────────────────────────────┘
```

#### 戻る操作

- スキル選択画面やターゲット選択画面で **「← 戻る」** をクリックするとコマンドメニューに戻れます

#### 属性相性（ダメージ 1.5倍 / 0.5倍）

| 攻撃属性 | 弱点（1.5倍） |
|------|------|
| 火 🔥 | 氷に有効 |
| 氷 ❄️ | 水に有効 |
| 雷 ⚡ | 火に有効 |
| 水 💧 | 雷に有効 |
| 光 ✨ | 闇に有効 |
| 闇 🌑 | 光に有効 |

#### 覚醒システム

- ダメージを受けると覚醒ゲージが溜まる（+15）
- 味方が戦闘不能になると大きく溜まる（+30）
- ゲージMAXで「覚醒」コマンドが光り、使用可能に

### 主要システム

- **ATBバトル** — アクティブタイムバトル（ウェイトモード）
- **属性相性** — 火・氷・雷・光・闇の5属性
- **覚醒スキル** — 覚醒ゲージMAXで発動する必殺技
- **星晶石** — 装備によるステータスカスタマイズ
- **フィールド探索** — 宝箱・セーブポイント・ランダムエンカウント
